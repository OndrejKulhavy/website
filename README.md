# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Project Structure

- `index.html`: Main HTML structure.
- `style.css`: Styles using CSS variables and modern layout techniques.
- `script.js`: Interactivity for mobile menu and scroll animations.

## How to Run

1.  Open the project folder in VS Code.
2.  Open `index.html`.
3.  Right-click and select "Open with Live Server" (if you have the Live Server extension installed) or simply open the file in your web browser.

## Customization

-   **Images**: Replace the placeholder texts in `index.html` with actual `<img>` tags pointing to your project screenshots and photos.
-   **Colors**: Edit the `:root` variables in `style.css` to change the color scheme.
-   **Content**: Update the text in `index.html` to reflect your latest experience and projects.
